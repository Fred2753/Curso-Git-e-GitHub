1 - Para criar uma nova branch
	git checkout -b "nome da branch"

	exemplo: git checkout -b "secundaria"

2 - para verficiar as branch e qual esta ativa
	git branch

3 - Para renomear uma branch
	a. Entra na branch
		git checkout "nome_branch_antiga"
	b. renomear a branch
		git branch -m "nome novo"

	exemplo:

	git checkout master

	git branch -m main

4 - git merge <nome branch que vai copiar>

	a. entra na branch que vai receber os arquivos e alterações realizadas na outra branch

	git checkout <nome da branch>
	
	b. realizar a copia (merge) da branch que tem as alterações.
	
	git merge <nome da branch>

	exemplo:

	git checkout main (entro na branch main)

	git merge secundaria (copio tudo da branch secundaria para branch main)

5 - Para deletar uma branch 
	git branch -D <nome branch>

	exemplo:

	git git branch -D secundaria
